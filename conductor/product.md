# Product Guide

## Initial Concept
An AI coding agent for the terminal with local LLM support, extended from Google's gemini-cli.

## Target Audience
- Teams needing a private/offline capable AI coding tool.

## Key Goals
- Provide a secure, offline-first alternative to cloud-based AI coding assistants.

## Core Features
- "Conductor" mode for structured, context-driven development tracks.

## Value Proposition (Conductor)
- Enables checkpoints and easy reversion of complex multi-step tasks, reducing risk in iterative development.

## Success Metrics
- User adoption of the "Conductor" workflow for complex feature implementation.
